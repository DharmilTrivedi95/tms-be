
## Technology Used

  

* Node.js

* Express

* Mongo
  

## Project setup

```
#1 install dependancy
- npm install

#2 copy or move env file
- cp .env.example .env
OR
- mv .env.example .env

#3 setup value in env file

#4 start project
- npm start 

```
